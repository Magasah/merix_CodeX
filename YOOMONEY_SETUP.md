## 🚀 Установка

### 1. Обновите код на сервере:

```bash
cd ~/merix_CodeX
git reset --hard origin/main
git pull origin main
```

### 2. Установите новую библиотеку:

```bash
pip3 install yoomoney==0.1.0
```

```bash

```

### 3. Перезапустите бота:

```bash
pkill -f main.py
nohup python3 main.py > bot.log 2>&1 &
```

## 🎯 Функциональность

### **1. Баланс пользователя**
- Автоматически добавляется колонка `balance` в таблицу `users`
- Новые пользователи регистрируются с балансом 0 RUB
- Баланс отображается в профиле: `💰 Баланс: {balance} RUB`

### **2. Пополнение баланса**
В профиле пользователя добавлена кнопка **"💳 Пополнить баланс"**

**Процесс пополнения:**
1. Пользователь нажимает кнопку
2. Вводит сумму (минимум 10 RUB)
3. Получает ссылку на оплату YooMoney
4. Оплачивает через банковскую карту
5. Нажимает "✅ Я оплатил"
6. Баланс автоматически начисляется после проверки

### **3. Проверка платежей**
- Используется YooMoney API для проверки статуса
- Уникальный `label` для каждого платежа: `user_{id}_{uuid}`
- Автоматическая верификация через `operation_history`

## 🛠️ Troubleshooting

### Ошибка: "ModuleNotFoundError: No module named 'yoomoney'"
```bash
pip3 install yoomoney
```

### Ошибка: "Invalid token"
- Проверьте токен в `config.py`
- Убедитесь что токен не истёк
- Получите новый токен на https://yoomoney.ru

### Баланс не начисляется
- Проверьте логи: `tail -f bot.log`
- Убедитесь что платёж имеет статус `success`
- Проверьте что label совпадает
