# 🌍 Merix CodeX Global Bot - Мультиязычный Telegram Бот

Полнофункциональный мультиязычный Telegram бот для IT-агентства **Merix CodeX** с базой данных SQLite и админ-панелью.

## 🎯 Основные возможности

### 🌐 Мультиязычность (i18n)
- **4 языка**: Русский 🇷🇺, English 🇺🇸, Тоҷикӣ 🇹🇯, O'zbekcha 🇺🇿
- Выбор языка при первом запуске
- Все тексты, кнопки и меню адаптируются под выбранный язык
- Легко редактируемый словарь переводов в `translations.py`

### 💾 База данных SQLite
- Автоматическое создание таблицы `users`
- Сохранение user_id, username, language, дата регистрации
- Отслеживание последнего взаимодействия

### 📂 Услуги
- **Telegram Боты**: от 200 TJS / 1800 RUB
- **Веб-сайты**: Landing pages & Corporate sites (цена обсуждается)
- **Безопасность/Pentest**: Basic Audit & Protection (цена обсуждается)
- **Официальный договор** для всех услуг
- **Поддержка 24/7**

### ✅ Система заказов (FSM)
1. Пользователь выбирает услугу
2. Описывает задачу
3. Подтверждает заявку
4. Заявка отправляется админу с полной информацией (ID, язык, описание)

### 🔐 Админ-панель (команда /admin)
- **📊 Статистика**: Количество пользователей в боте
- **📢 Рассылка**: Отправка сообщений всем пользователям
- Доступ только для ADMIN_ID

### 🆘 Помощь
- Прямая ссылка на админа (tg://user?id=ADMIN_ID)
- БЕЗ email адресов
- Круглосуточная поддержка

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка .env файла

Файл `.env` уже настроен с вашими данными:
```env
BOT_TOKEN=your_bot_token_here
ADMIN_ID=7679557111
PROXY_URL=
```

### 3. Запуск бота

**Вариант A: Напрямую (если есть VPN)**
```bash
python main.py
```

**Вариант B: С прокси (если Telegram заблокирован)**
1. Найдите рабочий SOCKS5 прокси
2. Добавьте в `.env`:
   ```env
   PROXY_URL=socks5://прокси:порт
   ```
3. Запустите: `python main.py`

**Вариант C: Используя bat-файл**
```bash
start_bot.bat
```

## 🗄️ База данных

### Структура таблицы `users`:
- `user_id` (INTEGER PRIMARY KEY) - Telegram ID
- `username` (TEXT) - Username пользователя
- `first_name` (TEXT) - Имя пользователя
- `language` (TEXT) - Код языка ('ru', 'en', 'tj', 'uz')
- `created_at` (TIMESTAMP) - Дата регистрации
- `last_interaction` (TIMESTAMP) - Последнее взаимодействие

### Просмотр базы данных:
```bash
sqlite3 bot_database.db "SELECT * FROM users;"
```

## 🐛 Решение проблем

### Ошибка подключения к Telegram API
**Решение**: Используйте VPN или прокси (см. раздел "Поддержка прокси")

### База данных не создается
**Решение**: Убедитесь, что у бота есть права на запись в папку

### Админ-панель не работает
**Решение**: Проверьте правильность ADMIN_ID в `.env`

### Переводы не работают
**Решение**: Проверьте структуру словаря `TRANSLATIONS` в `translations.py`
